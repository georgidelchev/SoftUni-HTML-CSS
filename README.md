## 📖 About

This repository contains my complete solutions for the **HTML & CSS** course at [SoftUni](https://softuni.bg) — a structured introduction to the building blocks of the web. The course progresses from writing valid semantic HTML through to crafting fully responsive layouts with Flexbox and Media Queries.

These fundamentals underpin every web project elsewhere in the portfolio — from the Razor views in [AYN](https://github.com/georgidelchev/AYN-) and [ViceIO](https://github.com/georgidelchev/ViceIO) to the frontend work in [CSharp-Web](https://github.com/georgidelchev/CSharp-Web).

Every solution has been submitted to the **Open Judge System (OJS)** and achieved a perfect **100/100** score.

## 📚 What This Course Covers

A complete front-end foundations curriculum — from the first `<html>` tag to fully responsive pages:

- **HTML fundamentals** — semantic elements, document structure, forms, tables, links, and media
- **CSS & Typography** — selectors, specificity, fonts, colours, text styling, and the cascade
- **CSS Box Model** — margin, padding, border, `box-sizing`, and layout fundamentals
- **Positioning** — `static`, `relative`, `absolute`, `fixed`, `sticky`, and float-based layouts
- **Flexbox** — one-dimensional layout, alignment, flex containers, and flex items
- **Responsive Design** — Media Queries, breakpoints, and mobile-first thinking

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/georgidelchev/SoftUni-HTML-CSS.git
   cd SoftUni-HTML-CSS
   ```

2. **Open any module folder** and launch `index.html` directly in your browser — no build step needed.

3. **Browse by topic** — navigate to the module folder matching the concept you want to explore.
